## 2026-06-05

### 渲染管理器优化
- 渲染时关闭 V-Sync + 解除帧率上限，大幅提升渲染速度
- 添加渲染分辨率参数 `set_res: Vector2i`（默认 1920x1080）
- 渲染完成后清除 `frame_*.png` 帧序列 + 恢复导入界面
- `import_screen` 响应 `render_finished` 信号重新显示

### 架构重构
- 主场景从 `world.tscn` 改为 `import_screen.tscn`
- World 改为动态 `instantiate()` + `add_sibling()` 加载
- `world.gd` 精简，`_ready()` 只做 JSON 解析 → `chart` 字典

### 音符系统
- 四种 Note（Tap/Drag/Hold/Flick）通过 `notes.gd` 的 `set_type()` 切换贴图
- Hold 特殊处理：独立 `hold.tscn` 场景，头/身/尾三段 `region_rect` 裁剪
- Hold body 缩放中心通过 `offset` 设在底部，`scale.y` 向上拉伸
- 多押高亮：`_mh` 变体贴图，ARGB → RGBA 颜色转换

### 打击特效系统
- `hit_effect.tscn` + `hit_effect.gd`：6x5=30 帧灰度图逐帧动画
- 颜色 `0xe1ffec9f`（ARGB）→ `Color("ffec9fe1")`（RGBA 金黄）
- 方块粒子：8 方向 × 5 种速度 = 40 个，0.93 衰减 + 线性透明度递减
- 0.5 秒后自动 `queue_free()`
- World 下 `effects_container` 挂载，`spawn_hit_effect(global_pos)` 调用
- 特效不跟随 note/判定线，取世界坐标独立放置

### 谱面格式参考
- Phigros chart JSON：`judgeLineList[].notesAbove/notesBelow`
- 时间单位 T：`1 T = 1.875 / BPM` 秒
- 坐标单位：X=108px/单位，Y=648px/单位
- 四种 note type：1=Tap, 2=Drag, 3=Hold, 4=Flick
